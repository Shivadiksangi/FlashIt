# FlashIt 🚀

## 📌 Project Overview

FlashIt is a web-based application developed using Django that provides users with an interactive and efficient platform. It is designed with a clean UI and structured backend to ensure smooth performance.

## 💡 Features

* User Authentication (Login/Register)
* Interactive UI using HTML, CSS, JavaScript
* Backend powered by Django
* SQLite Database integration
* Responsive design

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Django (Python)
* Database: SQLite

## 📂 Project Structure

* Templates (HTML files)
* Static files (CSS, JS, Images)
* Django backend files

## ▶️ How to Run

1. Install dependencies
2. Run:
   python manage.py runserver
3. Open browser: http://127.0.0.1:8000/
